## Research

**H-RSSM — Hyperbolic World Model** · [Project page](https://josselinperret.github.io/projects/h-rssm/)

Independent research project combining hyperbolic latent spaces with the Dreamer/RSSM architecture. The core idea is that many environments have inherently hierarchical structure that Euclidean latent spaces represent inefficiently; replacing them with Poincaré/Lorentz manifolds should yield better sample efficiency and compositional generalization. Implemented in JAX. Targeting a preprint.

**ZenkoLab** · [zenkolab.dev](https://www.zenkolab.dev)

Co-founder and CEO. Building robust diagnostic pipelines for ophthalmological imaging (diabetic retinopathy, AMD) — fine-tuning RETFound (ViT) on noisy clinical data, with post-hoc explainability. Grant recipient from Fondation CentraleSupélec.
